# [Salon22](https://salon22-c16e1.web.app/)

# Project Features

## User Context

- Login using Email/Google
- Book Service
- Order Review
- Order Cancel
- Check Status of Order (pending, confirm etc)
- Adding a review

## Admin Context

- Manage User
- Manage All Booking
- Manage All Services
- Add Service
- Add New Admin

# Tools & Technology Used (Client)

- React.js
- Firebase Auth
- Axios
- React router dom
- React-hook-form
- React-rating
- AOS-Animation
- Material UI
- Git Version Control + GitHub
- Firebase Hosting (Client Deploy)

# Tools & Technology Used (Server)

- Express.js
- MongoDB
- Heroku (Server Deploy)
- [Server](https://github.com/sawdahoque234/salon22-server)
